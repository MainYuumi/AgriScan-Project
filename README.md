# AgriScan-Project

A mobile application for **plant disease detection**.  
Currently, the application detects only **potato leaf diseases**.

---

## Features
- Detects diseases on potato leaves using images.
- Provides disease name and a link that forwards you to google for other details about the disease.
- Simple and user-friendly interface.

---

## How to Use
- Open the app on your phone
- Take a photo of a leaf or upload one
- The app will show the disease
